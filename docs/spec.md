# 仕様まとめ

## レイアウト

| 項目 | 内容                                          |
| ---- | --------------------------------------------- |
| 分割 | 左右分割、ロースタッガード                    |
| 左手 | 6列×5行 + EVQWGD001エンコーダー（escの左）    |
| 右手 | 7列×5行 + PMW3610トラックボール（親指エリア） |
| 配列 | US配列、FNキーなし、英数・かなキーなし        |

KLEレイアウトはキー数・キー順・ラベルの基準として扱う。Ergogen上の物理座標、列スタッガー、親指キー位置、トラックボール位置は1:1印刷プロトタイプで確認しながら調整する。

## KLEレイアウト

### 左手側

```json
[{a:5,f:5},"`\n~","!\n1","@\n2","#\n3","$\n4","%\n5"],
[{x:0.5,a:7,f:5},"tab","Q","W","E","R","T"],
[{x:0.75,a:4,f:2},"ctrl",{a:7,f:5},"A","S","D","F","G"],
[{x:1.25,a:4,f:2},"shift",{a:7,f:5},"Z","X","C","V","B"],
[{x:1.5,a:4,f:2},"esc","caps","opt","⌘","space"]
```

### 右手側

```json
[{a:5,f:5},"^\n6","&\n7","*\n8","(\n9",")\n0","\"\n'","+\n="],
[{x:0.5,a:7,f:5},"Y","U","I","O","P",{a:5},"{\n[","}\n]"],
[{x:0.75,a:7,f:5},"H","J","K","L",{a:4,f:2,w:1.5},"Enter","_\n-",{a:5,f:5},":\n;"],
[{x:1.25,a:7,f:5},"N","M","<\n,",">\n.","?\n/","↑","|\n\\"],
[{x:1.5,a:4,f:2},"space","⌘","opt",{x:0.75,a:7},"←","↓","→"]
```

## 入力デバイス

| 項目                   | 内容                                                |
| ---------------------- | --------------------------------------------------- |
| スイッチ               | Kailh Choc v1 Red（リニア）                         |
| トラックボールセンサー | PMW3610                                             |
| トラックボール基板     | SideraKB PMW3610 PCB Rev 2.x系互換                  |
| トラックボール径       | 34mm                                                |
| トラックボールクリック | Auto Mouseレイヤー（J・Kキー）                      |
| スクロール             | ボタン押しながらボール転がす                        |
| エンコーダー           | EVQWGD001（左手のみ）レイヤー0=音量、レイヤー1=輝度 |

## 接続・ファームウェア

| 項目           | 内容                                |
| -------------- | ----------------------------------- |
| ファームウェア | ZMK                                 |
| 左右間通信     | BLE                                 |
| PCとの接続     | BLEドングル（USB挿しっぱ）          |
| MCU            | nice!nano v2 × 3（左右 + ドングル） |

## PCB・ケース

| 項目                | 内容                                           |
| ------------------- | ---------------------------------------------- |
| PCB設計フロー       | Ergogen → FreeRouting → KiCad最終調整          |
| Ergogen初期成果物   | キー配置 + 外形 + 1:1印刷用アウトライン        |
| トラックボール領域  | 34mm円 + 46mm keepout円 + 23.5mm×31.5mm基板矩形 |
| PCB発注             | JLCPCB                                         |
| ケース（フェーズ1） | 3Dプリント（PLA/PETG）、DMM.make or JLCPCB発注 |

## キーキャップ

| 項目              | 内容                             |
| ----------------- | -------------------------------- |
| 1Uキー            | MBK市販品（印字なし、フェーズ1） |
| Enterキー（1.5U） | 3Dプリント自作（Choc v1ステム）  |
