# kbd-project

自分用の最高のキーボードを作るための、自作分割キーボードプロジェクト。
量産・販売は目的にしない。

## 概要

- 個人利用を最優先し、一般化・量産性・販売価格の最適化は優先しない
- 左右分割・ロースタッガード
- トラックボール付き（右手親指エリア）
- 完全無線（ZMK + BLEドングル）
- ロープロファイル（Choc v1）
