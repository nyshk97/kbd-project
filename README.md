# kbd-project

自分用の最高のキーボードを作るための、自作分割キーボードプロジェクト。
量産・販売は目的にしない。

## 概要

- 個人利用を最優先し、一般化・量産性・販売価格の最適化は優先しない
- 最終形は左右分割・ロースタッガード・トラックボール付き・完全無線を目指す
- 最初から最終形を作らず、簡単な試作を複数台作って段階的にリスクを潰す
- ファームウェアはZMK、MCUはXIAO nRF52840系、スイッチはKailh Choc v1を軸にする

## ディレクトリ構成

現在は、使うものだけを置く最小構成にする。

- `docs/` : プロジェクト全体の方針・ロードマップ・意思決定
- `boards/p01-simple-split/` : 初号機の目的、部品表、作業ログ
- `target/` : 最終的に目指す統合版
- `target/layout/` : 最終形のKLEレイアウトJSON
- `target/ergogen/` : 最終形のErgogen設定

まだ使わないディレクトリは作らない。必要になった時点で、各試作機や `target/` 配下に次を追加する。

- `layout/` : KLEや物理配列メモ
- `ergogen/` : Ergogen設定
- `pcb/` : KiCadファイル、GerberなどのPCB成果物
- `case/` : ケース3Dデータ、印刷用データ
- `firmware/` : ZMK設定ファイル
- `keycaps/` : キーキャップ3Dデータ

試作をまたいで再利用するものが出てきたら `shared/` を作る。外部資料や生成スクリプトが必要になったら `vendor/` や `tools/` を作る。
