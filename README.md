# kbd-project

自作分割キーボードのプロジェクトリポジトリ。

## 概要

- 左右分割・ロースタッガード
- トラックボール付き（右手親指エリア）
- 完全無線（ZMK + BLEドングル）
- ロープロファイル（Choc v1）

- [ ] ファームウェア
- [ ] キーキャップ自作
- [ ] 組み立て

## ディレクトリ構成

- `docs/` : 仕様・意思決定の記録
- `layout/` : KLEレイアウトJSON
- `ergogen/` : Ergogen設定ファイル
- `pcb/` : KiCadファイル
- `case/` : ケース3Dデータ
- `firmware/` : ZMK設定ファイル
- `keycaps/` : キーキャップ3Dデータ
